Multi Coloured Image

Prerequisites

  • Check you have java installed: issue java -version from the command line, it should be version 1.7 or above.
  • Check you have ant installed: issue ant -version from the command line, it should be version 1.6 or above.

How to

Install

expand the mci_bundle.zip file into a directory of your choice, it will be run from that directory.

Run

With ant: To produce a multi coloured image named poc.png in the images/ directory, issue the following command;
ant poc
To verify that an image contains only unqiue pixels, issue the following command;
ant verify -Dimage=name
Where image is the name of the image to be verified, it shall default to poc.png if not specified.

Without ant:

  • Create default image: java -jar dist/mci.jar
  • Create named image in specific directory: java -Dimage.dir=/volumes/x07/mydir -jar dist/mci.jar freddy.png - the order of the arguments is important!
  • Verify default image: java -classpath dist/mci.jar au.com.polly.mci.ImageVerifier poc.png

Run unit tests

Requires ant to be installed (I actually run the tests successfully within Intelli-J too). From the top level directory invoke the following command;
ant test

Manifest

  • readme.html - this file
  • mci.jar - classes containing the application
  • mci_test.jar - unit test classes and image files
  • images/ - test images
  • src/ - source code for the application
  • tests/ - source code for the unit tests
  • build.xml - ant build file
